This file contains different versions of the neighborhood information i found: 

1. 'neighborhoods.csv' --> the raw data extracted from QGIS
2. 'neighborhoods_cleaned.csv' --> the previous file filtered by just the valid columns
3. 'neighborhoods_enriched.csv' --> the previous file enriched with new features from QGIS (distance_from_{i}) + the feature 'Neighborhood' in English
4. 'neighborhoods_central.csv' --> the previous file filtered so it contains only neighborhoods from the 'Municipal Community of Volos'
