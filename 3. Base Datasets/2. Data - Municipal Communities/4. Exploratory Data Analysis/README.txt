In this folder we take the raw Economic and Demographic data for each Municipal Community and we process them: 

- I transformed some Demographic features from legit values to percentages
- I transformed some Demographic group features to less groups
- I transformed the Economic hierarchical features to new fraction features: 'unemployment_rate', 'labor_force_participation_rate', 
  'primary_sector_pct', 'secondary_sector_pct', 'tertiary_sector_pct'
- I added the feature 'Area_km2' (via QGIS)
- I addedd MANUALLY the English name of the Municipal Communities in 'ΔΗΜΟΤΙΚΗ ΚΟΙΝΟΤΗΤΑ' and renamed the old Greek name as 'ΔΗΜΟΤΙΚΗ ΚΟΙΝΟΤΗΤΑ ΕΛΛ'
