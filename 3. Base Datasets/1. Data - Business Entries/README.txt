This folder contains the whole information about acquiring the total 'base' business entries dataset: from scarping from multiple sources, to 
merging them into 1 dataset:

- 1. API Calls: I used the API of Foursquare to get business entries, and the API of Google Maps to help with Geocoding
- 2. Web Scraping: I used the 'Web Scraper Chrome Extension' to get data from multiple websites: Xrysos Odigos, Greek Catalog, Vacation Rental, and more
- 3. Merging - Business Data: the procedure of merging the business entries from all the sources (merging/cleaning/filling null values)
