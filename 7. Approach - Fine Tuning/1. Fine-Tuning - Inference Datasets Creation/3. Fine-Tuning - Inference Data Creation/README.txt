This folder contains information about the creation of the Fine-Tuning dataset from the Synthetic data I collected. These are
the steps I followed:

- I computed the Statistics of the Features so I can later provide justifications on the provided suggestions by
  comparing the individual feature value to these stats
- I created the Fine-Tuning dataset by using the Synthetic Categories for Fine-Tuning
- I created the Inference dataset by using the Synthetic Categories for Inference
