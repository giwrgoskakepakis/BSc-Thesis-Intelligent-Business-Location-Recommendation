This folder contains information about the approach of Fine-Tuning an LLM to perform the recommendations. 

1. I created the Fine-Tuning and Inference datasets
2. I Fine-Tuned an LLM on the Fine-Tuning dataset (3 attempts so far)
3. I inferenced the Fine-Tuned LLMs to extract their Inference Outputs
4. I used the Inference Outputs to Evaluate the LLMs and compare them

The best one since is the LLM that was Fine-Tuned on 5 Epochs!
